## Simple rules for a Pull Request everyone should follow:        
**Please NEVER EVER submit PRs with MAJOR overhalls (e.g, Another lang instead of YAML, major engine overall which forces YAML to be written differently, etc.)**     
**Please keep PRs on topic**    
**SUGGEST PRs of QOL, Security, Speed, just any non-major updates.**    
**This is a bit off topic, but if you want to fork, please have README with "Original version of WROKA was built by live-by-unix. Link to original repo is [here](https://github.com/live-by-unix/wroka).**    
## Thank you for following this, and if you do not, I you may have a lower chance of having a accepted PR. Please write in Rust.
